= Erfahren Sie mehr über ONTAP Support und Funktionen für Ubuntu Hosts
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP unterstützt die folgenden SAN-Hostfunktionen unabhängig von der auf Ihrem System ausgeführten ONTAP Version.

[cols="75,25"]
|===
| Funktion | Ubuntu-Host-Version 


| Das NetApp Plug-in im nativen Paket zeigt ONTAP-Details für NVMe/FC-Namespaces an.  a| 
22.04 oder später



| NVMe- und SCSI-Datenverkehr wird auf demselben Host auf einem bestimmten Hostbusadapter (HBA) unterstützt, ohne die expliziten dm-multipath-Einstellungen, um das Beanspruchen von NVMe-Namensräumen zu verhindern  a| 
22.04 oder später



| In-kernel-NVMe-Multipath ist für NVMe-Namespaces standardmäßig aktiviert. Daher ist keine explizite Konfiguration erforderlich  a| 
22.04 oder später

|===

NOTE: Einzelheiten zu unterstützten Konfigurationen finden Sie imlink:https://mysupport.netapp.com/matrix/["Interoperabilitäts-Matrix-Tool"^] .
